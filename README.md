# BraveLikeNewTab
Eine moderne Neuer-Tab-Startseite für Firefox mit lokal gespeicherten Hintergründen, Brave Search, Schnellzugriffen, Pins und Fokusmodus. Die Erweiterung arbeitet ohne externe Frameworks und ohne externe Favicon-Requests.

Du kannst eigene Hintergrundbilder per Datei-Upload oder Bild-URL hinzufügen, eigene Shortcuts anlegen und Icons lokal hochladen. Einstellungen, Bilder, Shortcuts und Pins werden lokal über browser.storage.local gespeichert.

## Nur in Deutsch (Only in German)

Jeder neue Tab hat ein neues Hintergrundbild.
Standartmäßige Suchmarschine ist Brave Search, kann aktuell nur im Source Code geändert werden.

Firefox: https://addons.mozilla.org/de/firefox/addon/brave-like-new-tab/
