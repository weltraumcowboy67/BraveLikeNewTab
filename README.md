# BraveLikeNewTab
Eine moderne Neuer-Tab-Startseite für Firefox mit lokal gespeicherten Hintergründen, Brave Search, Schnellzugriffen, Pins und Fokusmodus. Du kannst eigene Hintergrundbilder per Datei-Upload oder Bild-URL hinzufügen, eigene Shortcuts anlegen und Icons lokal hochladen. Einstellungen, Bilder, Shortcuts und Pins werden lokal gespeichert.
